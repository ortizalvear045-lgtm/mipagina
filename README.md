# Mi Página

Proyecto base de una landing page moderna para desplegar en Vercel.

## Estructura
- index.html: contenido de la página
- styles.css: estilos visuales
- script.js: comportamiento simple

## Despliegue en Vercel
1. Sube este proyecto a GitHub.
2. Entra a Vercel y crea un nuevo proyecto desde ese repositorio.
3. Vercel detectará automáticamente el sitio estático y lo desplegará.

## Desarrollo local
Puedes abrir el archivo index.html directamente o servir la carpeta con un servidor simple.
